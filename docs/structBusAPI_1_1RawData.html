Detailed Description

Struct that contains packet as an array of bytes (bus-level packet).

Data in this form is being sent and received from the bus. Data memory layout: [5 bits - id][3 bits - command][2 bytes - address][1 byte - length][<length> * 4 bytes - payload][2 bytes - crc]
